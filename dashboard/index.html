Quo Number Assignments

Each agent gets ONE of your 3 Quo numbers. When they click "Call" on a lead, the dashboard tells Quo to dial from that assigned number so caller-ID matches the agent's territory. Only 1 Quo seat is used (yours) — agents log into Quo with your shared credentials.

Flag a number

If one of your Quo numbers starts getting "Scam Likely" labels or low answer rates, click Flag next to it. Calls from that agent will show a warning banner on Call Command until you reassign them to a different number or clear the flag.
